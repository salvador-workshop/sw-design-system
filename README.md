## Development

MacOS Ventura 13.2.1

### Installation

```
npm install
```

### Running locally

```
npm run dev
```

Launches parcel (default URL: `http://localhost:1234`)

### Running storybook

```
npm run storybook
```

Launches storybook, for a deep dive into components.
